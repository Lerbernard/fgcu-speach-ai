# EGN 2111C — Course Description

Subject: EGN | Course: EGN 2111C

EGN 2111C
 Engineering Computer Graphics - 3 credits(s)
 Spatial visualization, descriptive geometry and design communication methodology. Computer graphics techniques used to conceive, model, analyze, simulate, and evaluate design ideas. Emphasis on state-of-the-art Computer-Aided design and geometric modeling software tools.
 Prerequisite(s)
