# CAP 4767 — Course Description

Subject: CAP | Course: CAP 4767

CAP 4767
 Exploratory Data Analysis - 3 credits(s)
 This course covers the main topics necessary to train a Data Analyst, that is, descriptive statistics, data visualization, and dashboard design. Overview and applications of Exploratory Data Analysis (EDA), data description and preparation, descriptive analysis, data visualization, data storytelling, and dashboard design.
 Prerequisite(s)
