# CGN 5716 — Course Description

Subject: CGN | Course: CGN 5716

CGN 5716
 Environmental Data Science - 3 credits(s)
 This course introduces water and environmental data analysis using Python, a dynamic programming language with powerful libraries for analyzing and visualizing water and environmental data that are essential for tackling real-world water resources and environmental challenges. Data from various sources will be explored such as NOAA, NASA, Copernicus, USGS, and Data.Gov, handling formats such as CSV, shapefile, and NetCDF. No prior programming experience is required.
