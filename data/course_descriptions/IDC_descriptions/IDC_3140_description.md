# IDC 3140 — Course Description

Subject: IDC | Course: IDC 3140

IDC 3140
 Intro to Data Science - 3 credits(s)
 This course provides an overview of data science, its main concepts, and applications. This includes, overview and applications, the data science workflow, data acquisition and preparation, descriptive analysis, data visualization, introduction to machine learning, and ethics in data science.
 Prerequisite(s)
