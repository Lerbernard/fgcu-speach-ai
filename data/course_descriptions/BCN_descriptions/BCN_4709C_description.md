# BCN 4709C — Course Description

Subject: BCN | Course: BCN 4709C

BCN 4709C
 Const Project Management - 3 credits(s)
 Introduction to project delivery methods and the underlying principles for choosing the appropriate system. Recognizing the complexity of the preconstruction process including conceptual estimating and scheduling, life cycle costing, constructability reviews, value engineering, risk management, and special contract requirements. Understanding management and administration of a construction project. Topics to be discussed include project initiation, planning, control and closeout; scope/quality, budget and schedule definition; team organization, and other issues throughout the project life cycle.
 Prerequisite(s)
