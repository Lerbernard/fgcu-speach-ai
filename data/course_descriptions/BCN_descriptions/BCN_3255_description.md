# BCN 3255 — Course Description

Subject: BCN | Course: BCN 3255

BCN 3255
 Graphic Communication in Const - 3 credits(s)
 Construction communication tools, including the use of computer-aided drafting (applications of 4-D modeling in construction), blueprint reading, free hand sketching, model building, piece-based simulations and the Internet.
 Prerequisite(s)
