# CIS 5377 — Course Description

Subject: CIS | Course: CIS 5377

CIS 5377
 Info. Security and Compliance - 3 credits(s)
 Covers design, analysis and implementation of basic security tools to protect sensitive data and information. Includes encryption, decryption, operation modes, hash function, public certificate, one-time pad, secure storage, verification of information source, secure random number generation, and vulnerabilities and attacks.
