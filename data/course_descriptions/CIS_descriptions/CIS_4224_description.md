# CIS 4224 — Course Description

Subject: CIS | Course: CIS 4224

CIS 4224
 Adv. Cybersecurity - 3 credits(s)
 Explores advanced cybersecurity concepts focusing on AI-driven threat detection and defense strategies. Design and evaluate intelligent intrusion detection systems (IDS), conduct penetration testing, assess blockchain, zero trust, and apply machine learning to automate cyberattack analysis and adaptive defense in network environments.
 Prerequisite(s)
