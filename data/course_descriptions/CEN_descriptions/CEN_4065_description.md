# CEN 4065 — Course Description

Subject: CEN | Course: CEN 4065

CEN 4065
 Software Architecture & Design - 3 credits(s)
 This course guides the students through the engineering process of building the software architecture and designing the software product according to rigorous design criteria. Additional emphasis is placed on the use of design tools and applications from the area of embedded real-time systems.
 Prerequisite(s)
