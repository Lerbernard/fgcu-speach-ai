CEN 4935
 SeniorSoftware Engr Project II - 3 credits(s)
 This is the second course in a year-long sequence of a senior software project. Students are expected to continue their software project from the first project course, i.e., Senior Software Engr Project I. Final report and presentation are among the deliverables.
