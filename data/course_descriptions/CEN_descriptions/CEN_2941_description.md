# CEN 2941 — Course Description

Subject: CEN | Course: CEN 2941

CEN 2941
 Intro Internship In Soft Eng - 0 credits(s)
 A 0-credit experiential learning course that provides students with the opportunity to gain practical exposure in software engineering. Students engage in supervised work or professional experience, paid or unpaid, that supports their development in software engineering. Participation requires active involvement in industry-related activities for the duration of the course.
 Prerequisite(s)
