# CEN 4026 — Course Description

Subject: CEN | Course: CEN 4026

CEN 4026
 Software Maint & Evolution - 3 credits(s)
 This course introduces students to the processes, methods, techniques, and tools in software maintenance and evolution. Topics include, but not limited to, concept location, change management and implementation, impact analysis, program comprehension, reverse engineering, re-engineering, and software analysis.
 Prerequisite(s)
