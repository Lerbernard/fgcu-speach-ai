# CEN 4216 — Course Description

Subject: CEN | Course: CEN 4216

CEN 4216
 Cyberphysical Systems - 3 credits(s)
 Students will learn fundamental concepts of designing and programming cyberphysical systems, understood as networked embedded computer systems, including their requirements specifications, architectural and detailed design, and implementation, focusing on real-time aspects of programming languages and operating system kernels. Learning the concepts will be enforced by a Project to design and develop a software application of practical importance.
 Prerequisite(s)
