# CAI 6107 — Course Description

Subject: CAI | Course: CAI 6107

CAI 6107
 Advanced Machine Learning - 3 credits(s)
 A broad introduction to advanced machine learning, including data pre-processing and feature selection; anomaly detection; advanced learning approaches (meta-learning, semi-supervised learning, reinforcement learning, federated learning, transfer learning, multimodal learning); biclustering; recommender systems; ensembles and committee machines; multilabel classification; unstructured data mining; graph and network mining.
